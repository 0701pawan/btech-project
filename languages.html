AQM. Air Quality Managmet

Hardware part


DHT11 sensor which is a temperature and humidity sensor, the data part of it is connected to the A0 pin of the Arduino.

MQ135 which sense gases like smoke, NOx, CO2, The data pin of the MQ135  sensor is connected to the A1 pin of Arduino.

MQ2 is used to sense smoke and methane . The data pin of the sensor is connected to the A2 pin of the Arduino.

After fetching the data from the sensors the data value is converted into JSON format so that it can be transferred to NODEMCU using serial communication.

5,6 pins of the Arduino is connected to the RX, TX pins of NODE MCU.

Node MCU is used to push data in the server.

In our project, we have used thingspeak server to show data in graphical form.

Software part


Data that is pushed on the server is used to visualize the data.

The website is created on which data is visualized.

The website is created using HTML,CSS,Javascript&Mapbox

It shows all the data of the different sensors using the website.

It has also a voice function that will be integrated into the website.
